Site Mobile WAP (Wireless Application Protocol)

Tu peux désormais visiter PWND Renard sur un téléphone cellulaire qui supporte le WAP (Wireless Application Protocol). Veuillez noter que le site mobile est toujours en construction et est présentement incomplet.

Entre ce lien URL sur ton cellulaire: www.pwndrenard.net/WAP/main.wml
